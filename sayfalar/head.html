Head Etiketi

<head> </head> elemanı tarayıcıya ve siteyi ziyeret eden arama botlarına site hakkında bilgi veren kısımdır.

head etiketi içierinde metadata ismi verilen bilgi verici etiketler bulunur. Bunlardan bazıları: <title>, <style>, <meta>, <link>, <script>, and <base> etiketleridir.

title

title etiketi tüm html sayfalarında olması gereken bir etikettir.
Bu etiket ile sekme başlığı, web sayfası vaforilere eklendiğinde gözükecek başlık ve arama motoru sonuçlarında gözükecek başlığı belirtir.

style

Bu etiket ile sayfa içerisinde kullanılmak üzere yazılan CSS kodları yazılır.

link

link etiketi ile sayfaya eklenecek olan harici css dosyası sayfaya bağlanır.
<link rel="stylesheet" href="mystyle.css">

meta

Meta etiketi ile sayfa hakkında özel bilgiler eklenir.Örneğin sayfanın hangi karakter kodu ile yazıldığ, sayfanın açıklaması, anahtar kelimeleri, yazar ismi.
meta etiketi ile eklenen veriler tarayıcıya içeriğin nasıl gösterileceği hakkında bilgi verir. Arama motorlarına anahtar kelimeler ve açıklama gibi bilgiler verir.

<meta charset="UTF-8">
<meta name="description" content="Free Web tutorials">
<meta name="keywords" content="HTML,CSS,XML,JavaScript">
<meta name="author" content="John Doe">

Viewport Ayarı

HTML'de web tasarımcılarının <meta> etiketi aracılığıyla görünüm alanı üzerinde kontrol sahibi olmalarını sağlayan bir yöntem vardır. Görünüm, kullanıcının web sayfasının görünür alandır. Cihaza bağlıdır ve cep telefonundaki bilgisayar ekranından daha küçük olacaktır. Tüm web sayfalarınıza aşağıdaki <meta> görünüm alanı öğesini eklemelisiniz:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Bir <meta> viewport öğesi, tarayıcıya sayfanın boyutlarını ve ölçeklendirmesini nasıl kontrol edeceğiniz konusunda talimatlar verir.
Width = device-width bölümü, aygıtın ekran genişliğini sayfanın genişliğini olarak ayarlar (aygıta bağlı olarak değişir).
initial-scale=1.0 bölümü, sayfa tarayıcı tarafından ilk yüklendiğinde başlangıç ​​zoom seviyesini ayarlar.

base

Bir sayfadaki tüm göreli URL'ler için temel URL'yi ve temel hedefi belirtir. Göreli URL ler normalde yazıldıkları sayfanın konumuna göre ayarlanmaktadır.
