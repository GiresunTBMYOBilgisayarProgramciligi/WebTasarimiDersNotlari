HTML Formlar

Web sayfalarında kullanıcılardan bilgi almak için formlar kullanılmaktadır. Kullanıcıdan alınan bilgiler, arayüzde anlık değişikliker yapmak için, web sunucusuna gönderilip işlenmek ve depolanmak için kullanılabilir.

HTML formu oluşturmak için <form> etiketi kullanılır. Form etiketi temel olarak action, method, name özniteliklerine ihtiyaç duyar.

Form elemanları

HTML formalar oluştururken input, select, textarea, button, fieldset ve legend, datalist etiketleri kullanılır.

input

Form oluştururken metin kutuları, onay kutuları, butonlar gibi bir çok eleman <input> etiketi ile oluşturulabilmektedir.
HTML5'te input etiketi ile oluşturabileceğimiz form elemanları aşağıda listelernmiştir.

  • <input type="button">
  • <input type="checkbox">
  • <input type="color">
  • <input type="date">
  • <input type="datetime-local">
  • <input type="email">
  • <input type="file">
  • <input type="hidden">
  • <input type="image">
  • <input type="month">
  • <input type="number">
  • <input type="password">
  • <input type="radio">
  • <input type="range">
  • <input type="reset">
  • <input type="search">
  • <input type="submit">
  • <input type="tel">
  • <input type="text">
  • <input type="time">
  • <input type="url">
  • <input type="week">

select

<sleect> etiketi ile aşağı açılır listeler oluşturulmaktadır.


selected özniteliğini eklediğimiz option etiketi seçili olarak gelir. Aşağıdaki açılır listede Lisans seçilidir.


Açılır listenin görünen eleman sayısını select etiketine eklediğimiz size özniteliği ile belirleyebiliriz.


Açılır listelerde birden fazla seçim yapmaya izin vermek için select etiketine multiple özniteliğini eklemeliyiz.Aşağıdaki açılır kutu ctrl tuşuna basılı tutarak bir den fazla seçime izin vermektedir.


textarea

Form içerisinde çok satırlı metin kutuları oluşturmak için <textarea> etikeli kullanılır.
textarea elemanının boyuru rows ve cols öznitelikleri ile yada css ile düzenlenebilir.

button

<button> etikeli ile form içerisine butonlar ekleyebilirsiniz. Butonlara tıklandığı zaman yapılacak işlemler javascript ile kontrol edilebilir.

fieldset ve legend

<fieldset> etiketi ile oluşturduğumuz formları bölümlere ayırabiliriz. <legend> etiketi ile de oluşturulan bu bölümlere başlık eklenmesini sağlar.

datailst

<datalist> etiketi, bir <input> elemanı için önceden tanımlanmış seçeneklerin bir listesini belirtir. Kullanıcılar, veri girerken önceden tanımlanmış seçeneklerin bir açılır listesini göreceklerdir. List özniteliği <input> öğesi, <datalist> öğesinin ıd özniteliği ile aynı olmalıdır.
